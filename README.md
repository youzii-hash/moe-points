# moe-points
这是一个wordpress的萌积分插件！！适用于二次元相关站点！

【萌积分】插件说明：

1：后台可以设置积分前缀，如：硬币：，金币：，积分：

2：设置发评论得到积分数量，（每篇文章限制只能通过评论获得一次）

3：被删除评论扣除积分数量

4：发布文章得到积分数量

5：兼容大部分投稿插件，通过投稿插件用户投稿的文章，会进入草稿箱或待审核，如管理员将草稿发布后，自动根据用户名为用户增加积分

6：设置新用户注册得到积分数量

7：设置每日通过发布文章获得积分上限

8：设置每日通过评论获得积分上限

9：设置每日登录自动获得积分数量

10：通过简码生成一个签到按钮，[qiandao]，填入文章或页面中显示签到按钮，设置通过点击签到按钮获得积分数量 24小时刷新还是每日定点刷新

11：等级头衔功能，如设置用户积分达到100，等级为2，头衔为入门魔法少女，默认的等级为1，头衔，新手魔法少女，（当然只是举例，可以通过后台自行设置，比如有几个等级，头衔名字，需要达到多少积分）

12：积分查看内容，设置简码
[ovo]隐藏起来的积分内容[/ovo]，后台可以设置被简码包裹的积分内容，需要用多少积分解锁

13：等级查看内容，设置简码
[666qwq]隐藏起来的积分等级内容[/666qwq]
后台可以设置被简码包裹的积分等级内容，需要达到哪个等级解锁，如选项框选取“入门魔法少女”，达到等级正常显示，未达到等级提示要哪个等级查看

14：积分排行+我的积分显示功能，自行后台设置显示前几名，以及排行榜名称，设置简码，[dengji]，
填入文章或页面中显示积分排行榜

15：悬浮球按钮功能，后台设置悬浮球的开关，开启后在右下角出现一个按钮，上滑动可隐藏，点击显示积分排行榜，个人积分数量，等级头衔

16：管理员后台可修改用户积分，并且有2个快捷的按钮，增加或减少用户积分

17：后台日志，可以显示所有用户积分变化记录，通过啥途径增加积分，以及在哪篇文章用了积分，可设置自动清理日志时间，1天，1个月，3个月

18：用户获得积分时有提示，如：签到成功，金币+3，评论成功，金币+3

19：卡密功能，可在发卡网出售，后台可以卡密数额，批量生成卡密，
设置简码，[duiovo]，
填入文章或页面中，把页面变成兑换卡密界面

20：前端个人主页功能，开启后用户登录默认返回首页，不进入wp的后台，支持前端更换头像，昵称，密码！以及个性签名！

21：新增头像框功能，支持为达到等级头衔的用户设置专属头像框！

22：文章显示作者功能，可在文章顶部或者底部设置显示作者小卡片，点击可访问作者个人主页~ (在文章评论区点击用户头像或者名字，一样可以访问用户个人主页)
